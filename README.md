#Proyecto-analisis de datos-empresa imaginaria
En este repositorio se desarrolara un mini proyecto de analisis de datos. 
Su utilizara como "materia prima" a una base de datos que simula ventas de una empresa de retail llamada "imaginaria", con multiples puntos de venta en argentina y en el exterior, asi como tambien por canal de comercio electronico.

El objetivo de este proyecto es identificar kpi's relevantes para la toma de decicones estrategicas, el control de gestion
